# Guns N' Roses Website

## Description
This is a page for the band Guns N' Roses, where you will find a section to buy tickets, read about them and see some pictures of their concerts

## Technologies used
- HTML5
- CSS3

## Project structure
```
├── imagenes
│   └── img-1.jpg ... img-10.jpg
├── css
│   └── style.css  
├── index.html
└── README.md
```
## How to run?
- Download or clone the repository.
- Open the project folder.
- Run the index.html file in a browser.
- Navigate through the different sections of the site.
